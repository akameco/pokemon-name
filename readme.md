# pokemon-name [![Build Status](https://travis-ci.org/akameco/pokemon-name.svg?branch=master)](https://travis-ci.org/akameco/pokemon-name)

> Get jp pokemon name


## Install

```
$ npm install --save pokemon-name
```


## Usage

```js
const pokemonNames = require('pokemon-name');

pokemonName('Pikachu');
//=> 'ピカチュウ'
```


## API

### pokemonName(name)

#### name

Type: `string`

Pokemon name
